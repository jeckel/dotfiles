# Tmux Cheat Sheet

## Navigation

### Panes
- `^b` + `-` : Split horizontaly
- `^b` + `_` : Split verticaly
- `^b` + Arrow Key : Navigate between splits windows
- `^b` + `o` : Swap panes
- `^b` + `q` : Show pane numbers
- `^b` + `x` : Kill pane

### Windows
- `^b` + `c` : Create new window
- `^b` + `w` : List windows
